rm				| 파일 삭제
	-f 	force 묻지마라
	-r	디렉토리까지 다 삭제
	-i	한번 더 물어보는 친구

[cp의 경우 ,i가 f보다 우선순위가 높다 	알림o]
cp				| 파일 복사
	-f 	force 묻지마라
	-i	한번 더 물어보는 친구
	-b	기존파일 ~붙이고 파일 만들기
	-r	강제로 복사
	
[mv의 경우 ,f가 i보다 우선순위가 높다	알림x]
mv				| 파일 이동(이름바꾸기)
	-f 	force 묻지마라
	-i	한번 더 물어보는 친구
	-b	기존파일 ~붙이고 파일 만들기	
	-v	상세 변경사항
